1.2 Intel GVT G project

GVT-G 是 Intel 的一个 open-source 的 GPU 虚拟化项目。

作为项目的作者、负责人之一的王叔叔，通过业余时间为开源社区做出了巨大贡献。

这次有幸请到了王叔叔为广大的开发者和 linux 技术爱好者，做免费的技术直播。

想为 linux kernel 社区提交patch的广大技术人员、 想拿到一个更好的实习机会的在校大学生、 想提高自己技术能力的在职工作人员、都可以参与进来。

Ft2 team 将会把 GVT-G 项目作为一个topic，学习研究。

感谢大家的参与。

欢迎访问 GVT-G 项目主页面 ===> Intel GVT-G project main page

1.3 ARM Neon study

从 ARMv4 开始，ARM就为指令集优化做了大量的工作，其中 SIMD指令，在V4、V5的时候，就有引入。

现在ARM指令集发展到了V7、V8，在一些ARM的介绍文档里面经常看到 Neon 这个词。Neon 是SIMD的一个升级版本。

对 ARM 系统架构、ARM指令集、ARM汇编有兴趣的同学，可以一起学习、探讨。

基于ARM平台的，优化工程师、嵌入式 AI、ML工程师，也可以参与进来学习、探讨。

Ft2 team 将会把 ARM neon 作为一个topic，学习研究。

感谢大家的参与。

欢迎访问 ARM neon 项目主页面 ===> ARM Neon main page
