1 Android build envrionment

  • Run docker ubuntu-12.04 hosted by ubuntu-16.04 
=> Docker install and run ubuntu 12.04 client
  sudo apt-get install docker.io
  sudo docker pull ubuntu:12.04
  cat /etc/issue # check ubuntu version
  sudo docker run -v /absolute/path/to/host:/absolute/path/to/guest -i -t ubuntu:12.04 /bin/bash
    -> Note: -v 后面的两个参数指定host挂载到client的两个目录，需要替换成你自己的实际目录。

=> install basic ubuntu 12.04 android envrionment
  Reference: http://10.46.8.32/doc_htc/build_env/ubuntu-12.04.html
  Note: Docker server 里面的 ubuntu 12.04 默认只有root用户，不过不要紧，这个root用户权限不高。
    按照手册安装12.04环境的时候，去掉sudo，安装之前先要 apt-get update 更新一下source。

=> install more ubuntu 12.04 android envrionment for docker

  apt-get install bc

  cp -r /path/to/repo/ /home  # cp your repo to docker ubuntu 12.04
  apt-get install vim
  vim ~/.bashrc  # set repo path
  source ~/.bashrc

  export USER=$(whoami)
  echo $USER  # just for check
  ln -s /usr/bin/gcc /usr/bin/cc
