1.2 often used command for sed

print

extract log 

Extract the lines which include the string from "2012-02-09" to "2012-09-10"
sed -n '/^2012-02-09/,/^2012-09-10/p' whole.log > part.log
reference

add string to line tail 

sed 's/$/tail/' test1.txt
